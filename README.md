## Typical Usage

##### Using Docker CLI
```
docker run -d --name bitlbee --restart=unless-stopped \
-v /persistent/appdata:/bitlbee-data \
-p 6667:6667 \
realies/bitlbee
```

##### Using Docker Compose
```
docker-compose up -d
```
